============================================
  OPTION 2: MANUAL SETUP
============================================

STEP 1: Create the API Key File
--------------------------------
Create a new file: server\.env

Add this content:
ANTHROPIC_API_KEY=your_actual_api_key_here
PORT=5000


STEP 2: Install Dependencies
-----------------------------
npm install
cd server
npm install
cd ..


STEP 3: Start the Servers
--------------------------
Open Terminal 1:
  cd server
  npm start

Open Terminal 2:
  npm start


STEP 4: Access the App
-----------------------
Frontend: http://localhost:3000
Backend:  http://localhost:5000

============================================
  TROUBLESHOOTING
============================================

Problem: JSON Parse Error / "Unterminated string"
Solution: User story is too long. Shorten it or break into parts.
         See TROUBLESHOOTING.md for details.

Problem: "API key not configured"
Solution: Make sure server/.env exists with your API key

Problem: "Cannot connect to backend"
Solution: Make sure backend server is running on port 5000

Problem: Still getting CORS errors
Solution: Check that backend is running and frontend is calling localhost:5000

Problem: Port already in use
Solution: Change PORT in server/.env to a different number
